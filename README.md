Ця кішка сіамська. Її звуть Мурка.

Якщо хвороба середня виводить на консоль:
“Хвороба середня. Потрібно лікування у клініці”

Хвороба легка. Лікуйте кішку дома.

Помилка. Введіть правильно ступінь хвороби.
